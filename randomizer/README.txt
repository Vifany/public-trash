
Bakamizer9000 v1.0.0

Программа копирует указанное количество случайных файлв с расширениями jpg, jpeg, gif, png и tiff из рабочей директории и подпапок в создаваемую в рабочей директории подпапку с именем "Randomized" без сохранения метаданных переименовывая их случайным образом.

Инструкция
1)Запустить
2)Указать количество файлов
3)Подтвердить способ генерации имён
4)???
5)ПРОФИТ!

Программа умеет генерировать имена по словарю, для этого необходимо поместить в рабочую папку файл словаря с именем dic.txt в кодировке utf-8 с элементами разделёнными переносом строки.
